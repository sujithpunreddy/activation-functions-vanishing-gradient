# Activation Functions and Vanishing Gradient Problem
This repository contains my machine learning tutorial which studies how different activation functions affect the learning behaviour of deep neural networks.
The main objective of this work is to understand how activation functions such as Sigmoid, Tanh and ReLU influence gradient propagation, training speed and classification performance.
The experiment is carried out using the Fashion-MNIST dataset. A fully connected neural network is trained multiple times by changing only the activation function in order to make a fair comparison.

Repository contents:
- Tutorial report (PDF)
- Jupyter Notebook implementation
- Training loss and validation accuracy plots
-
